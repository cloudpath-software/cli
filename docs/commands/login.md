# `login` (work in progress)

Login to your Cloudpath account
Opens a web browser to acquire an OAuth token.

**Usage**

```bash
cloudpath login
```

**Flags**

- `new` (*boolean*) - Login to new Cloudpath account
- `debug` (*boolean*) - Print debugging information
